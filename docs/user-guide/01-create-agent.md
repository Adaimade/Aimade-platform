# Step 1: Create Your AI Agent

## What is an Agent?

An Agent is your personalized AI bot. You define its name, personality, and what it can do.

## How to Create

1. Log in to the platform
2. Click **"+ New Agent"** in the top right
3. Fill in the form:
   - **Name** — what your bot will be called in Discord
   - **Description** — optional, for your own reference
   - **Skills** — pick what your bot can do (see below)
   - **AI Provider** — OpenAI, Gemini, or Anthropic
   - **Model** — e.g., `gpt-4o`, `gemini-1.5-pro`, `claude-3-5-sonnet`
   - **API Key** — your key for the AI provider (stored encrypted)
4. Click **Create Agent**

## Available Skills

| Skill | What it does |
|-------|-------------|
| Chat | AI replies when mentioned in Discord |
| Welcomer | Greets new members automatically |
| Moderation | Auto-mod, warn, kick commands |
| Polls | Create polls with `/poll` |
| Reminders | Set reminders with `/remind` |
| Music | Play music in voice channels |
